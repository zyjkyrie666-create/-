# 简历写法

## 中文版

**企业级 AI Agent 平台**

- 设计并实现面向企业场景的 AI Agent Runtime，支持任务规划、工具调用、结果汇总、审计追踪和运行状态持久化。
- 构建 Tool Registry 与 Policy Engine，对工具风险等级、角色权限和危险指令进行统一治理，降低 Agent 越权和误操作风险。
- 抽象 LLM Adapter，支持离线规则规划器与真实大模型接入，便于在本地演示和生产环境之间切换。
- 提供 FastAPI、CLI、Docker Compose 和自动化测试，形成可部署、可验证、可扩展的工程化交付。

## English Version

**Enterprise AI Agent Platform**

- Built an enterprise AI Agent runtime with task planning, tool execution, result synthesis, audit logging, and run persistence.
- Implemented a Tool Registry and Policy Engine to govern role-based access, tool risk levels, and unsafe instructions.
- Designed a pluggable LLM adapter layer supporting both offline rule-based planning and production LLM providers.
- Delivered FastAPI endpoints, CLI, Docker Compose, and automated tests to improve deployability and maintainability.
